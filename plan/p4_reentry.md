## 배경 — 무엇을 고치나 (§1 진단 5)

- **측정된 공백**: 청산 후 복귀 경로 부재. 60MA 청산 자체는 정당했으나(하이닉스 2025-08
  -28% 셰이크아웃 회피) 복귀 수단이 "새 5~7주 베이스 + 전 게이트"뿐이라, 시나리오 A는
  +2R로 하차 후 10배 구간을 방관했고 2026-04~06 +230% 구간(새 베이스 미성숙)은 구조적
  미참여였다. §1.3 시나리오 A(+2R) vs B(+30R)의 격차가 이 공백의 크기다.
- **철학 정합(§9)**: "재진입은 새 돌파 신호가 났을 때만"의 "신호" 확장이다. v4 초안 §9가
  명확화했듯 **§6② 추세 이탈로 이익 청산한 종목의 재진입은 복수 매매가 아니다** —
  손절 청산 후 재진입 금지는 그대로 유지된다.
- **전략 성격 변화 리스크(계획서 §9)**: 돌파 전략에 추세추종 재진입이 섞인다. 진입
  빈도·노출 증가 → MDD 상승 예상. 보조 진입임을 문언·집행(1차만·피라미딩 없음)으로 제한.

## 형태 (v4 초안 §4 매수 지점 ② — 이미 고정, 재확인)

- **자격 (전부 충족)**: (i) 직전 트레이드가 유효 돌파 진입 → **§6② 사유로 전량 청산**
  (`TREND_60MA_REST` 또는 `TREND_60MA_VOLBREAK`. 손절 청산이면 불가 — §9. 절반(HALF)·
  방어(③) 부분 매도는 "전량"이 아니므로 자격 사건이 아니다), (ii) 현재 무포지션,
  (iii) 트렌드 템플릿(§3 0단계) 통과, (iv) 시장 필터 정상.
- **트리거**: 청산 후 가격이 50일선(10주선)을 **종가 기준 상향 회복하고 M거래일 유지**
  하면 다음 날 진입. (M은 캘리브레이션 축, 초안 제안 2~3)
- **집행**: **1차 트랜치만(목표 비중의 50%), 피라미딩 없음.** 손절가 = 재진입 체결가 −
  2×ATR(−10% 캡) — 이전 트레이드의 평단·손절가와 무관한 **새 트레이드**. RS·과열·품질
  게이트는 요구하지 않는다(베이스가 없으므로 품질 게이트는 정의 자체가 불가; 자격
  (iii)(iv)가 초안 문언의 전부다).

계획서 §4 R4b 원문의 "50MA **부근에서 지지·반등**"은 v4 초안(나중 문언)에서 "종가 기준
**상향 회복 + 유지**"로 구체화됐다 — 눌림 접촉을 요구하지 않는 회복 확인형이 초안의
확정 형태이고, 본 설계는 초안을 따른다(더 단순하고 파라미터가 적다).

## Q6 세부 — 기계 판정 결정 질문 (제안 기본값, 승인 시 확정)

| # | 항목 | 제안 | 근거·대안 |
|---|---|---|---|
| Q6-1 | 회복 판정 | 자격 사건(전량 청산 체결일)부터, **실거래 바 세션의 종가 ≥ 50MA 연속 카운트**. 종가 < 50MA면 0으로 리셋. 카운트 ≥ M 도달일 d가 트리거 확인일 | M은 스윕 축 {2,3,5}. 청산 체결일 당일부터 기산(그날 종가부터 판정). 50MA 사전 하회 요구 없음 — 60MA 이탈 자체가 "이탈" 사건이고, 청산 시점 주가는 통상 50MA 아래라 사실상 자동 충족 |
| Q6-2 | 자격 유효 기간 W | 전량 청산 체결일부터 **W개월**(달력일 환산은 R3b와 동일 30.4일/월). 만료 시 자격 소멸 — 이후 복귀는 새 베이스 돌파(①)로만 | 스윕 축 {3,6,12}. 무제한이면 수년 뒤 우연 회복도 "추세 복귀"로 오인 — 새 사이클은 베이스가 맞다(R3b와 역할 분담) |
| Q6-3 | 횟수 제한 | **자격 1회 = 재진입 1회**(체결 시 소진). 재진입 트레이드가 다시 §6②로 전량 청산되면 **새 자격**(체인 허용, 명시 상한 없음 — 매 사이클이 자격·트리거 전부 재통과) | 발동 집계(§3.3)로 체인 빈도 감시. 트리거 확인 후 미체결(갭 상한)이면 자격 유지·재시도(아래 Q6-4) |
| Q6-4 | 체결 | 트리거 확인일 d 종가 판정 → **d+1 시가 체결**(청산과 동일 타이밍 계약). 상한 = **트리거 확인일 종가 +5%**(`entry.chase_limit_pct` 재사용): 시가가 상한 초과 갭이면 장중 저가가 상한 복귀 시 상한 체결, 아니면 미체결(1차 돌파 체결 문법 재사용). 미체결 시 카운트 ≥ M 유지 동안 **다음 세션 재시도**(확인일 종가 기준 갱신) | 시가 지정 없이 사면 런어웨이 갭에 무방비. 돌파 1차와 같은 체결 문법이라 코드 재사용·정합적 |
| Q6-5 | 게이트 판정 시점 | 트리거 확인일 **d 종가 기준**: 템플릿 `passes(d)`, 시장필터 `state_asof(d) == NORMAL` | 종가 결정·익일 체결 계약이라 d 종가 확정치 사용이 룩어헤드 없음. 돌파 진입의 ≤d-1 게이트와 기준일만 다르고 "판정 시점에 확정된 최신값" 원칙은 동일 |
| Q6-6 | 사이징·손절 ATR | 비중 = 기존 공식 min(20%, 1%/손절폭%) — 기준가 = **트리거 확인일 종가**, ATR = **확인일(d) 값**(체결일 d+1의 직전 세션 = d, 기존 "직전 세션" 계약과 동일). 수량 = 목표명목 × 1차 비율(50%) / 기준가 floor. 손절 = 체결가 − 2×ATR(d) (−10% 캡) | 재진입은 1차만이므로 실효 리스크 ≈ 0.5%(돌파 1차와 동일 구조) |
| Q6-7 | 자격 소멸 사건 | (i) 재진입 체결(소진), (ii) W 만료, (iii) **그 종목에 새 베이스 돌파 진입 발생**(신규 트레이드가 자격 대체), (iv) 새 §6② 전량 청산 발생 시 티켓 갱신(심볼당 최신 1개) | 손절·방어로 끝난 재진입 트레이드는 자격을 남기지 않는다(§9) |

재진입 포지션에 대한 규칙 적용(신규 트레이드로서 전부 동일): 손절 §6①(새 ATR)·60MA
§6②·시장 방어 §6③·8주 룰(재진입일 기준)·리스크 거버너 카운트 포함. **피라미딩 없음**
(거래량 게이트 무관, 2·3차 예약 없음). 베이스가 없으므로 단계(stage) 무관 — 트레이드
로그 `base_stage=0`, `pivot`=트리거 확인일 종가(기준가)로 기록.

## 엔진 확장 설계 (이벤트 루프)

하루 처리 순서에 두 단계 삽입 — 기존 1~6은 불변:

```
1.  대기 청산 체결 (전일 종가 결정 → 당일 시가)
1.5 대기 재진입 체결 (신규) — 전일 트리거 확인 → 당일 시가(+5% 상한, 갭 복귀 허용)
2.  장중 자동스탑 (대안 모델)
3.  피라미딩 2·3차 (장중)
4.  신규 돌파 진입 (장중)
5.  청산 판정 (종가)
5.5 재진입 판정 (종가, 신규) — 티켓별 회복 카운트 갱신 → 자격·게이트 통과 시 익일 대기
6.  자본곡선 기록
```

- **상태**: 엔진이 `dict[str, ReentryTicket]` 소유(심볼당 최신 1개). 티켓 = {자격 사건
  체결일, 만료일, 회복 카운터, 대기 여부}. 자격 부여는 `_record_sell`에서 전량 청산 +
  사유 ∈ {REST, VOLBREAK}일 때. 새 진입(`_try_open`) 성공 시 해당 심볼 티켓 폐기.
- **우선순위(자본 경합)**: 재진입은 시가 체결이라 **같은 날 장중 돌파 진입보다 자본을
  먼저 쓴다** — 타이밍의 자연 귀결(시가가 장중보다 먼저)로, 별도 우선순위 규칙을 두지
  않는다. 동일일 다중 재진입은 RS 내림차순·심볼 사전순(기존 결정론 규칙 재사용).
- **거래정지(결측 바)**: 판정은 실거래 바 세션만 카운트. 체결일에 바가 없으면 다음
  세션으로 이월(대기 청산과 동일).
- **거버너**: `new_trades_blocked`면 재진입 판정 자체를 보류(트리거 확인을 미루는 게
  아니라 그 날 대기 큐 진입만 차단 — 카운터는 계속 간다).
- **Q13 상호작용(명시)**: 재진입은 슬롯·현금을 소비한다 — 게이트 완화 누적으로 심해진
  자본 경합(Q13)을 더 조일 수 있다. 후보 실행에서 `portfolio_constraint` 병목 종목 수
  변화를 반드시 병기한다.

## 테스트 계획 (P1~P3 관례: 기본 꺼짐 비트 동치 + 신규 로직 유닛)

1. config 파싱 — enabled=true에 M/W 누락 시 ConfigError, false면 무시.
2. 자격 부여: REST/VOLBREAK 전량 청산만. STOP·부분 매도(HALF·방어)는 불가.
3. 회복 카운터: 연속 종가 ≥ 50MA 카운트, 하회 시 리셋, M 도달 → 익일 시가 체결.
4. 유효 기간: W 만료 후 트리거 불발. 새 돌파 진입 시 티켓 폐기.
5. 집행: 1차만(피라미딩 없음), 손절 = 체결가 − 2×ATR 캡, 갭 상한 미체결 → 재시도.
6. 게이트: 템플릿 실패·시장필터 비정상·거버너 중단 시 대기 차단.
7. 체인: 재진입 트레이드 §6② 전량 청산 → 새 자격. 손절 청산 → 자격 없음.
8. 룩어헤드 가드: 미래 바 조작이 그 이전 자본곡선 불변(기존 회귀 패턴 재사용).
9. **골든 다이제스트 불변**(enabled=false 기본) + 전 유니버스 off 행이 v3-5와 비트 동일.
