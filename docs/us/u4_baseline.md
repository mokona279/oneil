# U4 — 미국장 베이스라인 백테스트

**목표**: v4.0 무튜닝 이식(rules_us·costs_us)의 첫 공식 성적표. KR 트랙 p0와
같은 지위 — 이후 U5·U6의 모든 비교 기준점이 된다.

**선행**: U1(data_us 완비) + U3(코드 게이트 통과).

## 작업 항목 (순서 고정 — 골든 문화 이식)

1. **단일 종목 골든 3~5개** (`run_single`): 베이스·돌파가 차트로 알려진 사례
   (예: NVDA 2023 컵, AAPL 2019~2020 베이스, MSFT 등 — 선정 기준·차트 대조를
   기록에 남김). 판정 로그를 육안 검증 — 피벗·깊이·주수·돌파일이 차트와 부합
   하는지. **여기서 어긋나면 풀런 전에 원인 규명**(수정주가·value 근사·규칙 번역
   순으로 의심).
2. **전 유니버스 풀런**: rules_us.yaml·costs_us.yaml, U0 Q8 창(2017-01~최근),
   cash $100k → `out/us/p4/full/`. 유니버스 ~2배이므로 분리 프로세스 + 로그 +
   DONE 마커. **재현성: 동일 커맨드 2회 실행 비트 동일 확인.**
3. **캡처 세트 구축**: KR p0 방식 그대로 — +100%/252세션 달성 종목,
   20일 평균 거래대금 하한(U0 Q5 값), 티어(≥2/3/4/5×) — `out/us/p4/capture/`.
   빌더는 src/oneil_bt/analysis/capture.py 재사용(시장 중립 확인됨).
4. **sanity 체크**: 기간 내 유명 대박주(예: NVDA·SMCI·TSLA 등 — 목록을 먼저
   고정하고 결과를 대조)가 (a) 캡처 세트에 존재하는가 (b) 전략이 포착했는가
   (미포착이면 어느 게이트에서 탈락했는지 사유 분해 — KR Q14a 포렌식 방식).
5. **성적표**: 총수익·CAGR·MDD·승률·손익비·기대값(R)·캡처율(티어별)·
   슬롯 점유·비용 합계 + 벤치마크(S&P500·나스닥 매수보유) 병기.
   **생존편향 캐비앗을 모든 절대 수치에 병기**(KR과 동일 문화).
6. **고가주 스킵 빈도**(U0 Q11): 1주 > 슬롯 예산으로 미체결된 후보 수 집계.

## 산출물·DoD

- out/us/p4/ (골든·풀런·캡처·리포트)
- `plan/us/u4_baseline.md`: 골든 육안 검증 기록, 성적표, sanity 결과, 재현 커맨드
- DoD: 작업 1의 육안 검증 통과 + 풀런 재현 2회 비트 동일 + 성적표·캡처 완비.
  **성적이 나빠도 DoD와 무관** — 이 단계는 측정이지 판정이 아니다(판정은 U6).

  의미를 갖는다(U0 Q3 결정 병기). 골든 육안 검증에서 걸러진다.

  — 경계 ±10% 종목 수를 집계해 영향 규모를 기록.
