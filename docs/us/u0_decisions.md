# U0 — 사전 조사·결정 확정

**목표**: 미국장 적용의 모든 결정 질문을 조사 근거와 함께 표로 만들어 사용자
승인을 받는다. U1(수집)·U2(번역)의 입력이 전부 여기서 확정된다. **코드 작성 없음.**

**선행**: 없음 (docs/us/us_plan.md §3 인벤토리가 출발점)

## 작업 항목

1. **데이터 소스 비교표** (웹 조사 필수 — 기억으로 쓰지 말 것):
   후보 최소 Stooq / yfinance(Yahoo) / Tiingo / Alpha Vantage / EODHD / Norgate /
   Polygon. 비교 축: 일봉 범위(2015-10~ 커버), 수정주가 방식(분할만 vs 배당 포함),
   상폐 종목 포함 여부, 전 유니버스 일괄 수집 방법·소요, 증분 갱신 방법,
   호출 제한, 비용, 이용약관상 백테스트 사용 가부.
2. **SEC Section 31 수수료 연도별 이력 조사** (공식 fee rate advisory 기준,
   2015~현재) — costs_us.yaml의 `sell_tax_schedule` 계단으로 옮길 수 있는 형태로
   정리. FINRA TAF는 주당 과금이라 bp 환산 근사 or 무시를 제안과 함께.
3. **유니버스 정의 초안**: 거래소 범위(NYSE/NASDAQ/AMEX), 증권 유형 필터
   (보통주만 — ETF/ETN·우선주·워런트·유닛·SPAC 제외 방법), ADR 포함 여부,
   심볼 목록 원천(예: NASDAQ Trader symbol directory — 무료·현행 상장).
4. **결정 질문 표 작성 후 사용자 승인** (아래 목록 전부, 권장안 병기).

| Q9 | 비용: 커미션·슬리피지 | 커미션 0bp·슬리피지 5bp 유지 + U5 민감도(15/30/50bp) |
