# U6 — 한·미 비교 판정과 후속 결정

**목표**: U4·U5 실측을 KR 트랙(P0~P8)과 3층 기준(마스터 플랜 §5)으로 대조해
**전이 판정**을 내리고, 후속 트랙(재튜닝 / 규칙 재분류 / 운영 편입)을 사용자와
결정한다. 이 단계 산출물이 U트랙의 최종 문서다.

**선행**: U5 완료.

## 작업 항목

1. **한·미 비교표**: 총수익(벤치마크 대비)·CAGR·MDD·승률·손익비·기대값(R)·
   캡처율(티어별)·평균 보유기간·슬롯 점유·비용 비중 — KR(v4.0, +754.8% 경로)과
   US 베이스라인 병렬. 절대치 비교의 한계(생존편향·기간 레짐 차이) 서두 명시.
2. **축별 전이 판정표** (핵심): U5 분석 1의 델타 부호 대조를 3분류 —
   - **전이 성립**: KR·US 같은 부호 → 축의 일반성 강화. v4 규칙서에 병기.
   - **전이 실패**: 반대 부호 → KR 과적합 증거. P7이 "단일 레짐 플래그"로
     지목한 축(R3b·confirm5·slots12)이면 **KR 규칙서의 존속 근거 재분류를
     상정**(규칙 변경이 아니라 증거 재분류 — P8-3 방식).
   - **판정 불가**: 발동 저표본 → 표본 수와 함께 기록.
3. **기본 규칙 귀향 평가**: 트렌드 템플릿·베이스·8주 룰 등 US-native 규칙군이
   원산지에서 정상 작동했는가(캡처·sanity 결과로) — 실패 시 번역(U2)·데이터
   (U1) 결함 가능성을 먼저 소거했는지 확인.
4. **후속 결정 상정** (사용자 승인 대상):
   - US 전용 재튜닝 트랙 개설 여부(개설 시 KR P1~P5와 동일한 스윕·검증 문화)
   - KR 규칙서 갱신 여부(축 재분류 — 2의 결과에 따라)
   - 운영 편입 여부: screen_today·포워드 원장의 US 버전(별도 계획 필요, 실계좌
     전제 조건 포함)
   - 생존편향 보정 데이터(유료) 재검토 여부
5. **마스터 플랜 §5 3층 판정문**: 각 층 통과/부분/실패 + 근거 요약.
